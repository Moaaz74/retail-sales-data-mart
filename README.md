# Retail Sales Data Mart

## Project Overview

This project aims to design and implement a dimensional data mart for a retail sales dataset using Kimball dimensional modeling techniques.

The project includes:

* Business requirements analysis
* Data profiling and exploration
* Dimensional modeling
* Star schema design
* ETL development
* Data quality validation
* Analytical reporting

## Dataset

The dataset contains retail sales transactions, customer information, product details, shipping information, and geographical data.

## Objectives

* Design a scalable dimensional model
* Build a star schema
* Implement ETL pipelines
* Create analytical datasets
* Generate business insights
